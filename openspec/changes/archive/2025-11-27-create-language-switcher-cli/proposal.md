# Proposal: Создание CLI утилиты для локализации OpenSpec

## Summary
Создать CLI утилиту на Node.js, которая позволяет модифицировать файл `openspec/AGENTS.md`, заменяя стандартные инструкции на локализованные версии (например, на русском языке). Утилита будет запускаться через `npx`.

## Problem
OpenSpec по умолчанию генерирует инструкции на английском языке. Пользователи, желающие работать с AI на другом языке (например, на русском), вынуждены вручную переводить или адаптировать промпты.

## Solution
Разработать CLI инструмент, который:
1. Принимает аргумент с кодом языка (locale).
2. Загружает соответствующий шаблон промпта.
3. Обновляет `openspec/AGENTS.md` в текущем проекте.

## Impact
- Упрощение настройки OpenSpec для неанглоязычных пользователей.
- Автоматизация процесса смены языка инструкций.
