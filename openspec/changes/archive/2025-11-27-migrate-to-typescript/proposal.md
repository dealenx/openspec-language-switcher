# Proposal: Миграция на TypeScript

## Summary
Перевести кодовую базу проекта с JavaScript на TypeScript и реализовать полноценный процесс сборки через `npm run build`.

## Problem
Текущий проект написан на JavaScript, что не обеспечивает статической типизации. Команда `npm run build` является заглушкой и не выполняет полезной работы.

## Solution
1. Внедрить TypeScript и необходимые типы.
2. Перенести исходный код в директорию `src/` и конвертировать в `.ts`.
3. Настроить компиляцию в директорию `dist/`.
4. Обновить скрипт `build` для запуска компилятора `tsc`.

## Impact
- Повышение надежности кода за счет статической типизации.
- Появление реального этапа сборки проекта.
- Изменение структуры проекта (исходники в `src/`, артефакты в `dist/`).
